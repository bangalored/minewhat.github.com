MineWhat API

MineWhat Wiki to API Usage

  • Set Product Inventory

    Send us inventory details for all the products. It will help us provide you with better analysis on what to keep on in your inventory so that you never run out of stock or pile it up. MWSDK.inventory(product_id, count);

    product_id The unique identifier of the product of the product you want us to track.

    count The count of items remaining in the stock

  • Set Customer Email

    Users can clear cache and cookies from browser. Help us associate logged in users with their multiple session online by sending user email when he signs in. MWSDK.setUser(email_id);

    email_id Email Id of the signed in customer

  • Set Order Status

    Users might be visiting your website to check the order status. We can guide while they are at it. Send us order status details when a user lands on your page. MWSDK.orderStatus(email_id, order_id, order_status);

    email_id Email Id of the signed in customer

    order_id Order Id of the signed in customer

    order_status Order Status of the Order Id
